Custom Search Plugin adds a dynamic auto-suggestion search bar to your WordPress site, allowing users to search for posts by title.

## Description

This plugin enhances your WordPress site with a custom search feature that provides users with auto-suggestions as they type in the search bar. The suggestions are based on post titles, making it easier for users to find relevant content.

## Features

- Dynamic auto-suggestion search by post title.
- Customizable through shortcode.
- Simple and lightweight.

## Installation

1. Upload the `custom-search-plugin` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

## Usage

Add the `[custom_search]` shortcode to any post or page where you want the custom auto-suggestion search bar to appear.

```plaintext
[custom_search]
